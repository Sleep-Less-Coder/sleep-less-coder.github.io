Patient Management System (PMS)

Patient Management System (PMS) is an WindowsÂ® Application software which I developed as the project work while I was doing my BScIT (Bachelor of Science in Information Technology).

Patient Management System (PMS) is a simple, intuitive, robust and high performance, database driven software which can be used in health related institutions like clinics, nursing homes, hospitals, and the likes for managing their patients. Being a management software , it has a host of features like registering new patients, editing patient details, deleting, searching patients, management of medicines, reports generation of patients, etc just to name a few. It is a multiple user software which can be used by the front-desk staff, doctors, pharmacists and administrator with their respective login credentials and do the tasks that matters to them. The main logic behind the software is to resolve the issue plaguing most medical institutions of Nepal by enabling the software technology to handle the details of the patients/track the patients from the time they enter medical institution until they leave without requiring to lift up the pen and paper ever.

Modern and proven technologies are adopted for the development of the project. The main application logic is handled by C#.NET programming language and the database is served by the MySQL database server.
